# 软件使用流程

## 导入数据

+ 将PLC中的报警全部复制到一个txt
+ 将触摸屏里面的报警导出
+ 然后选择读取两个数据

## 对比数据

+ 点击解析数据，取出上标和下标
+ 点击对比数据，生成对比日志和保存数据
+ 点击保存，会**新建一个Excel**，将其手工复制到触摸屏导出的

# 优化体验

+ 对于-1的数据，应该填入Group-1方便识别的同时还能少填一点

# 二次保证

+ 使用前准备个表格
+ A列直接粘贴上威纶通原来的报警列
+ B列粘贴上软件修改后的报警列
+ 然后在Excel点击顶部菜单栏「开始」→「条件格式」→「新建规则」。在弹出的窗口里，选择 **「使用公式确定要设置格式的单元格」**。在公式框中输入：`=$A1<>$B1`。点击「格式」→「填充」→ 选择红色 → 点击「确定」。再次点击「确定」，所有 A、B 列内容不一致的单元格，会自动被标红。
